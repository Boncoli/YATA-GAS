# YATA Project TODO & Ambitions

YATA (boncoli RasPi) をさらに進化させ、真の「パーソナル・AI・パートナー」へと昇華させるためのロードマップ。

## 🚀 最優先の野望 (The Grand Ambition)
### 1. 記憶と指示の同期 (Shared Memory)
- [ ] **NAS (DS220j) への GEMINI.md 統合**:
    - 各デバイス（RasPi, メインPC, etc.）でバラバラになっている指示や長期記憶を NAS 上の一つのファイルに集約する。
    - 各デバイスからシンボリックリンクまたは同期スクリプトを介して参照し、「どこで話しても一人の私」を実現する。

## 📊 次なるライフログ収集 (Next Logging)
- [ ] **身体ログ (iPhone HealthKit)**:
    - 歩数、睡眠、心拍数を取得し、ドライブログの時間軸と統合。
    - 「ロングドライブと疲労度」の相関を分析可能にする。
- [ ] **推しログ (Hololive / YouTube)**:
    - 推しの配信時刻や投稿を自動取得し、自分の行動履歴と同じ地図/タイムライン上に並べる。
- [ ] **デジタル資産ログ (Steam / NAS)**:
    - ゲームプレイ時間や HandBrake のエンコード状況を RasPi に集約。
- [ ] **ホームエネルギーモニタリング**:
    - スマートプラグ等と連携し、自宅の消費電力を可視化。Ryzenマシンの「本気」とラズパイの「省エネ」をグラフで対比。

## 🧠 AI・インテリジェンスの深化
- [ ] **RSSオートチューニング**:
    - ユーザーが「よく読む記事」をAIが学習し、不要なフィードの無効化や新しいフィードの追加を自律的に提案。
- [ ] **パーソナルWiki (Obsidian等) 連携**:
    - AI要約の中でも特に価値の高い情報を、個人の知識ベースへ自動転送。
- [ ] **ローカル音声合成 (TTS)**:
    - 朝のDiscord要約を、ラズパイが自室や車内で読み上げ。

## 🗺️ 地図の進化 (Map Evolution)
- [ ] **タイムライン・フィルタリング**:
    - 指定した期間（月単位、連休単位）の軌跡だけを表示する機能。
- [ ] **ルート・ハイライト**:
    - 地図上の線をクリックして、そのドライブの詳細（日付、タイトル、距離）を表示。
- [ ] **ヒートマップ表示**:
    - よく通る道（CX-80の活動圏内）を視覚的に強調。
