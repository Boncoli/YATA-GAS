# YATA Project TODO & Ambitions

YATA (boncoli RasPi) をさらに進化させ、真の「パーソナル・AI・パートナー」へと昇華させるためのロードマップ。

## 📊 次なるライフログ・パーソナライズ (Next Evolution)
- [ ] **デジタル・ツイン (分身) の構築と深化**:
    - 「独り言キャッチャー」で得た生の思考プロセスを Gemini で分析し、BON 様の価値観や判断基準をプロファイリング。
    - ニュースや事象に対し「BON 様ならどう考え、どう反応するか」をシミュレート・代弁できるレベルまで精度を上げる。
- [ ] **興味関心ベースの「意味的マッチング」向上**:
    - 現在のキーワード＋AI要約ピックアップをさらに強化。ベクトル検索を使い、キーワードが直接一致しなくても「文脈的に近い」ニュースを確実に捉える。
- [ ] **独り言キャッチャー (Voice-to-Text ToDo)**:
    - ラズパイ5にマイクを接続し、日常のつぶやきや独り言を文字起こしして Gemini で分析・自動登録。
- [ ] **Apple Watch 連携の安定化**:
    - 外出先からの Watch 単体送信の成功率向上（リトライ処理の実装）。

- [x] **ダイナミック・ペルソナ・エディタの実装 (2026/02/27)**:
    - ポータルサイト（`portal.html`）から `persona.txt` を直接編集・保存できるUIを構築。
    - 気分に合わせて「性格」を即座に切り替えられるようにした。
- [ ] **ローカル音声合成 (TTS)**:
    - 朝のDiscord要約を、ラズパイが自室や車内で読み上げ。

## 🗺️ 地図の進化 (Map Evolution)
- [ ] **タイムライン・フィルタリングの高度化**:
    - 指定した期間の軌跡のみを表示し、その期間の総走行距離や平均燃費も併記する。
- [ ] **ヒートマップ表示**:
    - よく通る道（CX-80の活動圏内）を視覚的に強調。
