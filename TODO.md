# YATA Project TODO & Ambitions

YATA (boncoli RasPi) をさらに進化させ、真の「パーソナル・AI・パートナー」へと昇華させるためのロードマップ。

## 🚀 最優先の野望 (The Grand Ambition)
### 1. 記憶と指示の同期 (Shared Memory)
- [ ] **NAS (DS220j) への GEMINI.md 統合**:
    - 各デバイス（RasPi, メインPC, etc.）でバラバラになっている指示や長期記憶を NAS 上の一つのファイルに集約する。
    - 各デバイスからシンボリックリンクまたは同期スクリプトを介して参照し、「どこで話しても一人の私」を実現する。

## 📊 次なるライフログ収集 (Next Logging)
- [ ] **給油時・即時フィードバック通知**:
    - `fuel-log` 受信時に、即座に今回の区間燃費を計算して Discord へ「今回の燃費は ○○ km/L でした！」と通知する。
- [ ] **独り言キャッチャー (Voice-to-Text ToDo)**:
    - ラズパイ5にマイクを接続し、日常のつぶやきや独り言を文字起こしして Gemini で分析・自動登録。
- [ ] **身体ログ (iPhone HealthKit)**:
    - 歩数、睡眠、心拍数を取得し、ドライブログの時間軸と統合。
- [ ] **Apple Watch 連携の安定化**:
    - 外出先からの Watch 単体送信の成功率向上（リトライ処理の実装）。

## 🧠 AI・インテリジェンスの深化
- [ ] **RSSオートチューニング**:
    - ユーザーの購読傾向をAIが学習し、フィードの入れ替えを自律的に提案。
- [ ] **ローカル音声合成 (TTS)**:
    - 朝のDiscord要約を、ラズパイが自室や車内で読み上げ。

## 🗺️ 地図の進化 (Map Evolution)
- [ ] **タイムライン・フィルタリングの高度化**:
    - 指定した期間の軌跡のみを表示し、その期間の総走行距離や平均燃費も併記する。
- [ ] **ルート・ハイライト**:
    - 地図上の線をクリックして、そのドライブの詳細を表示。
- [ ] **ヒートマップ表示**:
    - よく通る道（CX-80の活動圏内）を視覚的に強調。

## 🔧 メンテナンス・基盤 (Maintenance)
- [ ] **PROJECT_ARCHITECTURE.md の保守**:
    - 新機能（ネイティブ燃費ダッシュボード等）を反映し、構成図を最新に保つ。
