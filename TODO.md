# YATA Project TODO & Ambitions

YATA (boncoli RasPi) をさらに進化させ、真の「パーソナル・AI・パートナー」へと昇華させるためのロードマップ。

## 🚀 最優先の野望 (The Grand Ambition)
- [ ] **給油時・即時フィードバック通知**:
    - `fuel-log` 受信時に、即座に今回の区間燃費を計算して Discord へ「今回の燃費は ○○ km/L でした！」と通知する。

## 📊 次なるライフログ・パーソナライズ (Next Evolution)
- [ ] **興味関心ベースの「意味的マッチング」向上**:
    - 現在のキーワード＋AI要約ピックアップをさらに強化。ベクトル検索を使い、キーワードが直接一致しなくても「文脈的に近い」ニュースを確実に捉える。
- [x] **クリック履歴に基づく関心プロファイリングの自動化**:
    - Discord投稿前の自動更新システムを構築 (2026/02 完了)。
- [ ] **独り言キャッチャー (Voice-to-Text ToDo)**:
    - ラズパイ5にマイクを接続し、日常のつぶやきや独り言を文字起こしして Gemini で分析・自動登録。
- [ ] **身体ログ (iPhone HealthKit)**:
    - 歩数、睡眠、心拍数を取得し、ドライブログの時間軸と統合。
- [ ] **Apple Watch 連携の安定化**:
    - 外出先からの Watch 単体送信の成功率向上（リトライ処理の実装）。

## 🧠 AI・インテリジェンスの深化
- [ ] **RSSオートチューニング**:
    - ユーザーの購読傾向をAIが学習し、フィードの入れ替えを自律的に提案。
- [ ] **ローカル音声合成 (TTS)**:
    - 朝のDiscord要約を、ラズパイが自室や車内で読み上げ。

## 🗺️ 地図の進化 (Map Evolution)
- [ ] **タイムライン・フィルタリングの高度化**:
    - 指定した期間の軌跡のみを表示し、その期間の総走行距離や平均燃費も併記する。
- [ ] **ヒートマップ表示**:
    - よく通る道（CX-80の活動圏内）を視覚的に強調。

## 🔧 メンテナンス・基盤 (Maintenance)
- [ ] **PROJECT_ARCHITECTURE.md の保守**:
    - 新機能（ネイティブ燃費ダッシュボード等）を反映し、構成図を最新に保つ。

## ✅ 完了した実績 (Achievements)
- [x] **三位一体の Shared Memory (NAS) 統合 (2026/02/16)**:
    - Win, Mac, RasPi の全筐体で `GEMINI.md` を NAS (DS220j) 経由で同期する仕組みを構築。
    - 相互バックアップスクリプト（PowerShell/Bash）を配備し、データ整合性チェック（ファイルサイズ監視）の掟を記憶に刻印。
- [x] **リポジトリ構造の「正本と現場」の分離 (2026/02/16)**:
    - `main` ブランチをロジック正本（GAS互換）のみに整理し、`local-raspi` を運用の現場として定義。整理成果をリモートへ完全プッシュ。
- [x] **SD極限保護 (Lazy Commit) の導入 (2026/02/15)**:
    - `run-ram.sh` に `--no-sync` を実装し、書き込み回数を1/6に削減。
- [x] **ネイティブ燃費ダッシュボードの実装 (2026/02/15)**:
    - Grafanaに依存せず、iPhoneから最高に見やすい燃費レポート画面 (`fuel.html`) を構築。
- [x] **インテリジェント・チャタリング防止ロジック (2026/02/15)**:
    - 家（Wi-Fi）の不安定な接続を自動修正する「隙間埋め/通過排除」と、車のクイックな行動記録を両立。
